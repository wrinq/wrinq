The making of wrinq

Things that go on behind the scenes to make wrinq the best tool that helps you streamline your rental property business.

Definitely an openresty guide

wrinq is built using openresty, a packaging of nginx giving the user the capability to program, rather than just configure, it with an easy to use programming language lua. We found openresty an invaluable tool to make fast, but still light on resource, web applications. So we decided to make a little guide to help someone new to openresty to get up to speed quickly.

Easy user accounts management with couchdb

In which I describe how to make couchdb handle the registration of new users and then manage their sessions in just three HTTP requests. After reading this you will never have to write any boilerplate code for user registration and session management again.

Testing couchapps with couchdb

In which we use couchdb's show and list to test couchdb design doc functions. Also present a step by step process to make CABS:- Couchapp Automated Build Systems. Very handy.

How we simulate user behaviour to test for errors

We use erlang's capability to model state machines using the built in otp libraries of gen_fsm and supervisors to simulate the user behaviour. Every possible action and every possible state is thoroughly tested before a new feature is shipped. All this so you can have a quality product in your hands.

At wrinq our mission is to help landlords run their property rental business efficiently. We send automatic reminders to your tenants as the due date of rent draws near. We collect your rent via paypal and deposit it into your paypal account. We automatically issue rent slips on your behalf. And we keep your books up to date. Wrinq makes everything more convenient for you and your tenants. The result is you attract better tenants, who pay on time. Start using wrinq and see your rental business grow.
