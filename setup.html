Setting up wrinq

  1. Download and install redis,erlang and nginx
  2. Start redis and nginx.
  3. clone wrinq-core
  4. cd to clone location and then type rebar get-deps compile followed by relx in terminal
  5. navigate to _rel folder and type bin/wrinq console. The application should start
  6. clone wrinq-front end
  7. Use the nginx.conf supplied in the repository. Edit the root directive to point to the location where you cloned wrinq-front-end. Reload (nginx -s reload) nginx
  8. Point your browser to localhost:3000
  9. The application should start now.
You might want to check some features of wrinq
